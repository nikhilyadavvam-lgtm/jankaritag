Yeh NGF132 Sampark website 3 main UI design types use kar rahi hai:

1. MINIMALIST DESIGN TYPE
Clean white background

Plenty of whitespace

Simple elements

Focus on readability

Uncluttered layout

Use: Header, feature sections, product cards

2. FLAT DESIGN TYPE
Simple colored icons (yellow car, green phone, pink document)

No 3D effects or shadows on icons

Bold solid colors

Geometric shapes (rounded rectangles)

Use: Feature icons, accent elements

3. MODERN/CONTEMPORARY DESIGN TYPE
Hero section with cinematic 3D background image

Typography-focused (bold headings)

Limited color palette (White + Yellow + Gray)

Card-based layout (product cards)

Clear visual hierarchy

Use: Overall website structure
